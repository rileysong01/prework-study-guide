HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML attributes are always defined in the beginning element
  • We use semantic HTML elements to indicate the role of their content & give structure to the content of the webpage

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Each declaration must contain the CSS property we want to apply + value of the property
  • * declare all elements of the page will have the style properties indicated

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add all changes in the current working brnach to the staging area for the next commit
  • git commit -m "commit message": commit created that captures a snapshot of the currently staged changes
  • git pull origin main: receieve the main branch into the local environment (always perform before a git push)
  • git push origin "branch-name": push local changes in branch-name to GitHub

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • console.log is a method used to display messages or values in the browser's console
  • Strict Equality Operator (===) checks to see if two values are equal and returns a Boolean result (true/false)
  • Arrays are single variables used to hold a group of data
  • Index is a number specificing the location of an individual data item inside an array (starting with 0)
  • for loop is a function that uses the predictable pattern of indeces to perform a task on all the items in an array by allowing a single code block to be executed on loop
